# 📊 Sales Performance Dashboard | Power BI

An interactive Power BI dashboard designed to analyze sales performance and uncover key business insights.

## 🔍 Key Insights
- Identified top-performing products and categories driving revenue
- Analyzed revenue trends over time to detect peak periods
- Compared order volume vs revenue to highlight high-value products
- Evaluated order status distribution and customer behavior patterns

## 📈 Dashboard Features
- KPI Cards (Total Revenue, Orders, Avg Order Value, Tax & Freight)
- Revenue Trend Analysis (Line Chart)
- Top 10 Products by Revenue
- Product Performance Analysis (Scatter Chart)
- Order Status Distribution (Donut Chart)
- Revenue by Product (Area Chart)

## 🛠 Tools & Technologies
- Power BI
- DAX
- Power Query
